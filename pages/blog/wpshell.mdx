# Create server 🍭
まずはいつも通りcentos7.9でサーバーを作成してください

入力する場所と内容は以下の通りです

|アーカイブ|管理ユーザーのパスワード|ホスト名|名前|タグ|
| - | - | - | - | - |
|CentOS 7.9(2009) 64bit|自由|学籍番号|学籍番号|学籍番号|

上記の入力が終わったら、サーバー作成をしてください

この時、時価が9円か12円になっていることを確認してから作成してください

# curl🐶
コンソールにログインして下記を入力

``` shell
curl https://raw.githubusercontent.com/Jarigyani/weather-app/main/front/src/assets/dcwp.sh > ~/dcwp.sh
```

# 実行

👇のセレクトボックスから自分の学籍番号を選択して、生成されたコマンドを実行してください

<Wpshell />

> 実行すると色々出ますが放置して次の手順に進んでください
> 学籍番号等を間違えたまま実行してしまった場合はcontrol+cで一度停止して、再度修正したコマンドを実行してください
