## Create server 🍭
まずはいつも通りcentos7.9でサーバーを作成してください

入力する場所と内容は以下の通りです

|アーカイブ|管理ユーザーのパスワード|ホスト名|名前|タグ|
| - | - | - | - | - |
|CentOS 7.9(2009) 64bit|自由|学籍番号|学籍番号|学籍番号|

上記の入力が終わったら、サーバー作成をしてください

この時、時価が9円か12円になっていることを確認してから作成してください

## DNS setting ✈️
👇画面から自分のサーバーのIPアドレスをDNSのレコードに設定してください

![](/blog/wpshell/1-min.png)

## 実行

コンソールにログインし、👇のセレクトボックスから自分の学籍番号を選択して、生成されたコマンドを実行してください

> 先ほど自分で設定したDNSレコード名を選択してください<br />基本は左のセレックとボックスから学籍番号を選択すればOKですが、独自に設定した場合は右のテキストボックスに入力してください

<Wpshell />

> 学籍番号等を間違えたまま実行してしまった場合はcontrol+cで一度停止して、再度修正したコマンドを実行してください

## 起動確認😎

コマンドの完了まではしばらく時間がかかります

ブラウザで <Wpshell2 /> にアクセスしてしばらく待ち👇みたいになってたら完了
