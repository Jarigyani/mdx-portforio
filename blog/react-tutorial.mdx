## nodeのインストール 📦
再起動するとターミナルからvoltaコマンドが使えるようになっているので、WindowsTerminalを起動し、下記コマンドでnodeをインストールする

```bash
volta install node@18.12.1
```

## ESLintのインストール 🚨

インストールして初期設定を行う

```bash
npm install eslint eslint-plugin-react-hooks --save-dev

npm init @eslint/config
```

完了すると.eslintrc.ymlが作成されるので、下記のように変更する

```yaml
env:
  browser: true
  es2021: true

extends:
  - eslint:recommended
  - plugin:react/recommended
  - plugin:react-hooks/recommended

overrides: []

parserOptions:
  ecmaVersion: latest
  sourceType: module

plugins:
  - react
  - react-hooks

rules:
  react/prop-types: off
  react/react-in-jsx-scope: off
  react-hooks/exhaustive-deps: warn
  react-hooks/rules-of-hooks: error

settings:
  react:
    version: detect
```

package.jsonに下記を追記する

```json
"scripts": {
	"lint": "eslint . --ext .js,.jsx"
}
```

下記コマンドでlintを実行する

```bash
npm run lint
```

エラーが出たら修正する

## Prettierのインストール 🎨

インストールして初期設定を行う

```bash
npm install --save-dev prettier eslint-config-Prettier
```

.eslintrc.ymlに下記を追記する

```yaml
extends:
	- eslint:recommended
	- plugin:react/recommended
	- plugin:react-hooks/recommended
	- prettier
```

プロジェクト直下に.prettierrcを作成し、下記を追記する

```js
{
  "tabWidth": 2,
  "singleQuote": true,
  "trailingComma": "none",
  "semi": false,
  "useTabs": false
}
```

## emotionのインストール 🎨

今回はスタイリングにCSS-in-JSのemotionを使う

まずはemotionをインストールする

```bash
npm install @emotion/react
```

次に、vite.config.jsの設定を変更する

```js
plugins: [react({
  jsxImportSource: '@emotion/react',
})]
```
eslintrc.ymlの設定も変更する

```yaml
rules:
	react/prop-types: off
	react/react-in-jsx-scope: off
	react-hooks/exhaustive-deps: warn
	react-hooks/rules-of-hooks: error
	react/no-unknown-property: [error, {ignore: [css]}] # for emotion
```

## VSCodeの設定 🖥

最低限入れておきたい拡張機能をインストールする

- [Prettier - Code formatter](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)

- [Live Share](https://marketplace.visualstudio.com/items?itemName=MS-vsliveshare.vsliveshare)

- [vscode-styled-components](https://marketplace.visualstudio.com/items?itemName=styled-components.vscode-styled-components)

- [Material Icon Theme](https://marketplace.visualstudio.com/items?itemName=PKief.material-icon-theme)

### 設定の変更 🛠

setting.jsonに下記を追記する

```json
"[javascript]": {
	"editor.formatOnSave": true,
	"editor.defaultFormatter": "esbenp.prettier-vscode"
},
"[javascriptreact]": {
	"editor.formatOnSave": true,
	"editor.defaultFormatter": "esbenp.prettier-vscode"
},
```
