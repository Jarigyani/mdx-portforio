### コード書いてく

環境構築は省く

```python
import discord
import os
from dotenv import load_dotenv

load_dotenv()

TOKEN = os.environ.get("TOKEN")

bot = commands.Bot(command_prefix="!", intents=discord.Intents.all())


@bot.event
async def on_ready():
    print("Bot is ready.")

bot.run(TOKEN)
```

この状態で動かすとターミナルに`Bot is ready.`と表示される
