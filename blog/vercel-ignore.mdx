## Ignore build stepをプロジェクトルートに作成 🍙

```shell
touch vercel-ignore-build-step.sh
```

```shell
#!/bin/bash

echo "VERCEL_GIT_COMMIT_REF: $VERCEL_GIT_COMMIT_REF"

if [[ "$VERCEL_GIT_COMMIT_REF" == "ビルドしたいブランチ名" ]] ; then
  # Proceed with the build
  echo "😘 - Build can proceed"
  exit 1;

else
  # Don't build
  echo "🥶 - Build cancelled"
  exit 0;
fi
```
